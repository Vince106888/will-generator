# Roadmap: Next Steps

This sequence focuses on finishing the core will drafting flow and preparing for staged rollout.

1. Complete API integration for the drafting flow (persist drafts, return generated IDs).
2. Expand complexity and validity engines with Kenyan-specific rules and escalation triggers.
3. Add save/resume and signing guidance workflows.
4. Harden PDF output with legal review and formatting polish.
5. Add authentication and secure draft storage (optional for MVP, required for scale).
6. Add staging deployment and data retention/backups.
7. Improve test coverage for critical flows and API contracts.
8. Add production deployment with approval gates.
